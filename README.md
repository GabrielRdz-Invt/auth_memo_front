# 🚀 Auth Memo App – React + Vite

https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB  
https://img.shields.io/badge/Vite-646CFF?style=for-the-badge&logo=vite&logoColor=white

This is a **React** application built with **Vite**, designed for the **Finance Department** to create and manage **Authorization Memos**.

The frontend allows users to:
- Create authorization memos  
- Attach related contracts or documents  
- Send requests through an approval workflow  
- View memo details and approval status  

> 🏢 Internal project that streamlines administrative processes and reduces manual document handling.
